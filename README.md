# 🤖 Witty Groq Chatbot

A modern, fast, and sarcastically smart chatbot interface built using the GenAI stack.

## 🚀 Features
- **High-Speed Inference:** Leverages `llama-3.3-70b-versatile` via Groq.
- **Witty Persona:** Custom System Prompt engineering for a unique user experience.
- **Modern UI:** Built with Streamlit for a clean, responsive chat interface.
- **Session Memory:** Remembers conversation context within the session.

## 🛠️ Tech Stack
- **Language:** Python
- **Orchestration:** LangChain
- **LLM:** Llama 3.3 (Groq)
- **Frontend:** Streamlit
- **Deployment:** Streamlit Community Cloud
